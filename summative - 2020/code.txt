			<p>
		  		For the summative project, we will create a GUI and modified version of the board game, "Guess Who?", using JQuery. "Guess Who?" is a very popular board game where each player is given a board with 24 mystery faces however in this game, it will only include 20. At the beginning of the game, the player will get an opportunity to select a character that the computer will have to guess and vice versa. The objective is to guess the "mystery" person on your opponents card by asking a yes or no question per turn about things that could help you determine their mystery character. For instance, let's say you ask if their mystery person has blonde hair and they respond with no, all the characters will blonde hair will be removed. This in turn, will eliminate faces depending on what you asked and if it was correct or not, making the board smaller and smaller and the game closer and closer to the end. This will go on until one person has guessed correctly, making them the victor, however, you get ONE guess. If the guess is wrong, you lose!

This screen will also include a play button, which will show the user the game screen with the board of characters. In this game screen, the user will be shown a help message that will briefly tell them to select a character and will be shown the game board. After the user has selected their character by clicking on their icons, they will be shown another help message that tells them to choose a question from the board. Once the user has gone and the characters are removed, a pop up of the computer's turn will appear with options of "Yes" and "No" for their question. Once the user or computer is left with one character, they will make a guess and a pop up will appear on the screen with the user or computer's guess. If the guess is correct, the win screen will show, otherwise, the lose screen will show. Both of these screens will have a button to return to the menu. There will also be multiple error screens that will pop up if the user does something they are not supposed to do such as hitting play while not having their name entered.
